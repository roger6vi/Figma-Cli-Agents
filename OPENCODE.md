Read `AGENTS.md` for full Figma CLI usage, commands, and syntax reference.

## Session Context

Check these environment variables for current session state:
- `FIGMA_DESKTOP_APP` — selected app (`stable` or `beta`)
- `FIGMA_PROJECT_DIR` — project-specific directory for scripts, exports, and skills
- `FIGMA_SAFE_MODE` — `true` if running in Safe Mode (plugin-based, no patching)
